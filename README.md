# Useful links

[Install python](https://www.python.org/downloads/)

[Pandas docs](https://pandas.pydata.org/docs/user_guide/index.html)

# Add libraries

```sh
# Windows
pip install matplotlib
pip install pandas

# Mac OS
pip3 install matplotlib
pip3 install pandas
```

# Run application

```sh
# Windows
python main.py

# Mac OS
python3 main.py
```
